# NPC Sprite Sources

Create one lowercase `snake_case` directory per NPC or reusable NPC design:

```text
npcs/<npc_slug>/sprite_sheet.png
```

Record direction and frame order in an adjacent asset `README.md` when it is not obvious. The
internal demo's current standard walk layout is three frames across four rows ordered Down, Left,
Right, Up, but supplied sheets may use another explicitly documented layout.
