# Pokemon Engine

A custom Pokemon game engine built with Electron, TypeScript, and Vite.

## Getting Started

Follow these instructions to get the project up and running on your local machine.

### Prerequisites

- [Node.js](https://nodejs.org/) (Ensure you have a recent version installed)

### Installation

1. Clone the repository (if you haven't already).
2. Install the project dependencies:

   ```bash
   npm install
   ```

### Running the Project

To launch the application in development mode with hot-reloading:

```bash
npm run dev
```

## Building for Production

To build the application for production:

```bash
npm run build
```

## Project Structure

- **src/**: Contains the source code for the Electron main process and renderer.
- **scripts/**: Utility scripts for data processing and testing.
- **data/**: JSON files storing game data (items, moves, etc.).
- **maps/**: Map data files.
