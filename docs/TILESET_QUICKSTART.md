# Quick Start Guide - Tileset System

## Testing the Tileset System

I've created a sample tileset for you to test with!

### Step 1: Import the Sample Tileset

1. **Open the game** in Editor mode (`npm run dev`)
2. **Switch to "Tilesets" tab**
3. **Click "Import New Tileset"**
4. **Enter the following:**
   - Name: `Sample Tileset`
   - Tile Size: `32`
   - Path: `data/tilesets/images/sample_tileset.png`

### Step 2: Start Painting

1. **Go to "Map" tab**
2. **You'll see a grid of tiles** from the tileset
3. **Click any tile** to select it
4. **Click on the map** to paint!

## The Sample Tileset Includes:

- **Grass tiles** (various shades)
- **Dirt paths**
- **Water tiles** (animated frames)
- **Rocks and stones**
- **Trees and plants**
- **Flowers**
- **Building walls** (brick)
- **Roof tiles**
- **Doors and windows**
- **Fences**

## Adding Your Own Tilesets

1. **Place your PNG tileset** in `data/tilesets/images/`
2. **Import via the editor** using the same process
3. **Make sure your tileset** is organized in a grid
4. **Common tile sizes:** 16x16, 32x32, 48x48

## Tips

- The tileset should be a perfect grid (no gaps)
- All tiles should be the same size
- PNG format works best
- You can have multiple tilesets and switch between them

Enjoy building your Pokemon world!
